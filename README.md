# Sales Performance Dashboard (Excel Project)

This project is an Excel-based Sales Performance Dashboard designed to analyze and monitor the performance of sales executives. The dashboard uses a dataset containing sales representatives, their total sales, targets, and performance gaps.

The objective of the project is to provide a clear overview of how each sales executive is performing relative to their assigned targets. By visualizing key metrics, the dashboard helps identify top performers and employees who are falling behind their targets.

The dataset includes information such as sales executive names, total sales achieved, target values, and the percentage difference from the target. This allows managers to quickly evaluate overall sales productivity.

Using Excel tools such as pivot tables, formulas, charts, and dashboard elements, the project transforms raw sales data into meaningful business insights.

The dashboard helps organizations track sales efficiency, monitor performance trends, and support data-driven decision making.

This project demonstrates practical skills in **Excel analytics, dashboard creation, business reporting, and data visualization**.

Tools used: **Microsoft Excel (.xlsm), Pivot Tables, Charts, Data Analysis, and Dashboard Design.**
